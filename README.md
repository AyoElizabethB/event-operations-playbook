# Event Operations Playbook

A high-level structure for managing complex international events from planning to reconciliation.

## 1. Stakeholder Alignment
- Define success metrics
- Confirm budget ownership
- Establish approval hierarchy
- Risk & compliance assessment

## 2. Budget Management
- Multi-stream tracking
- Forecast vs actual reconciliation
- Supplier negotiation & cost control

## 3. Vendor & Agency Management
- Clear scope of work
- Defined milestones
- Production timelines
- Onsite roles & escalation paths

## 4. Delegate & Tech Integration
- Registration platform integration
- CRM sync
- Automated confirmation workflows
- Reporting dashboards

## 5. Post-Event Analysis
- Operational review
- Financial reconciliation
- Workflow improvements
- Repeatable process documentation

This playbook reflects my structured approach to delivering scalable and commercially accountable event programmes.
